## 🚀 Sobre o Projeto
O site da **AGL Company** é uma plataforma moderna e responsiva desenvolvida para apresentar soluções de Inteligência Artificial de ponta. O projeto foca em uma experiência de usuário (UX) premium, utilizando tecnologias web puras para demonstrar como a IA pode transformar dados brutos em decisões inteligentes para empresas.

> [!IMPORTANT]
> Este é um **projeto acadêmico** desenvolvido para fins educacionais como trabalho escolar.

## 🏢 A Empresa
A AGL Company é uma empresa fictícia especializada em serviços de tecnologia e IA de última geração. Nossa missão é automatizar processos, gerar insights profundos e acelerar o crescimento de negócios através de:
- **IA Preditiva:** Antecipação de tendências de mercado.
- **NLP Avançado:** Processamento de linguagem natural para atendimento e análise.
- **Visão Computacional:** Reconhecimento e automação baseada em imagens.
- **Arquitetura de Dados:** Infraestrutura robusta para modelos de escala.
